# Evolv Labs Website

A modern, sleek website for Evolv Labs - a health innovation studio focused on solving real problems for real people.

## Features

- **Modern Design**: Sleek, contemporary UI with smooth transitions and animations
- **Responsive**: Fully responsive design that works on all devices
- **Smooth Animations**: High-quality transitions and scroll animations
- **Glassmorphism**: Modern glassmorphic design elements
- **Gradient Effects**: Beautiful gradient backgrounds and accents
- **Interactive Elements**: Hover effects and smooth scrolling

## Technologies Used

- HTML5
- CSS3 (with modern features like backdrop-filter, CSS Grid, Flexbox)
- Vanilla JavaScript (no dependencies)
- Google Fonts (Inter)

## Getting Started

1. Simply open `index.html` in your web browser
2. No build process or dependencies required

## Structure

- `index.html` - Main HTML structure
- `styles.css` - All styling and animations
- `script.js` - Interactive functionality and animations

## Sections

1. **Hero** - Eye-catching introduction with animated background
2. **About** - Company description and key features
3. **Work** - Showcase of products and events
4. **Team** - Founder profiles
5. **Contact** - Links to LinkedIn and website
6. **Footer** - Additional information and links

## Customization

The website uses CSS custom properties (variables) defined in `:root` in `styles.css`, making it easy to customize colors, spacing, and other design elements.

## Browser Support

Works on all modern browsers that support:
- CSS Grid
- Flexbox
- CSS Custom Properties
- Backdrop Filter
- Intersection Observer API
